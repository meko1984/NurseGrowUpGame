# 看護師道場-新人成長物語-

看護判断の学習ゲーム「看護師道場-新人成長物語-」の初期プロトタイプです。スマートフォンの縦画面を優先し、HTML / CSS / JavaScriptのみで作成しています。

## 起動方法

`index.html` をブラウザで開いてください。ビルドや外部ライブラリは不要です。

## 今回実装した内容

- スマホ向けメイン画面
- ユニット病室マップ
- 4床表示（肺炎、心不全、消化管出血、空床）
- タップ地点へのプレイヤー移動アニメーション
- ベッド・設備のオブジェクトタップ
- スタッフ用PC画面と患者一覧
- 4床分のセントラルモニター表示
- 注意アラーム表示
- 移動ごとにゲーム内時刻を5分進める処理
- 仮クエストと達成メッセージ
- PCでも利用できるレスポンシブ表示

## 未実装の内容

- 本格的なクエスト
- スコア、点数、評価、ランキング
- 成長システム
- 保存機能
- 患者状態変化
- 結果画面

## 注意事項

このゲームは看護判断の学習を目的としたフィクションです。実際の医療行為・判断は、所属施設の手順、医師の指示、最新のガイドラインに従ってください。
